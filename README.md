# Personal_Health_and_Lifestyle_Analyser

**The Personal Health & Lifestyle Analyzer is a Python-based application that helps users track and evaluate their daily health habits. It allows users to record important lifestyle data such as sleep hours, water intake, steps walked, screen time, and number of meals.

The program stores this data and analyzes it to provide a health score for each day. This score is calculated based on healthy lifestyle standards like getting enough sleep, drinking sufficient water, staying physically active, limiting screen time, and maintaining proper eating habits.

The application also provides:

📅 Daily Summary – Displays the health score for each recorded day
📊 Weekly Summary – Calculates average values of all health parameters
💡 Suggestions – Gives improvement tips based on user habits

This project demonstrates the use of Python fundamentals such as:

Lists and dictionaries (for data storage)
Functions (modular programming)
Loops and conditions (logic building)
User input handling

Overall, this tool helps users become more aware of their lifestyle and encourages healthier daily routines.**
